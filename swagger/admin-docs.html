üîê Autentikasi Required

Semua endpoint memerlukan autentikasi JWT token. Klik tombol "Authorize" di atas dan masukkan token dalam format:

Bearer your_jwt_token_here

Cara mendapatkan token: Login melalui POST /api/auth/admin/login dengan kredensial admin yang valid.

üë• Level Akses Admin

  • Super Admin (Level 1): Akses penuh ke semua endpoint dan fitur sistem
  • Admin OPD (Level 2): Akses terbatas ke SKPD yang ditugaskan
  • Admin UPT (Level 3): Akses terbatas ke Satker yang ditugaskan

Note: Dokumentasi ini menampilkan semua endpoint Super Admin. Admin OPD dan UPT memiliki subset endpoint sesuai level akses.
